# Babanga

Data-driven healthcare intelligence and emergency department optimization.

## Mission

Babanga is building tools that help hospitals make better operational decisions through forecasting, analytics, and recommendation systems.

Our initial focus is emergency department staffing and patient demand prediction, helping healthcare organizations improve resource allocation, reduce wait times, and optimize staffing decisions.

---

## Current Project

### Emergency Room Staffing Recommendation System

An intelligent decision-support platform that combines healthcare, environmental, and public data sources to:

* Forecast emergency department demand
* Predict patient intake volume
* Recommend staffing levels
* Improve resource allocation
* Reduce operational inefficiencies

### Data Sources

Current research areas include:

* Historical hospital occupancy and wait times
* EMS dispatch data
* Weather data
* Flu and respiratory illness trends
* Public holidays and local events
* Homelessness data
